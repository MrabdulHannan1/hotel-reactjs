## Run Locally

Clone the project

```bash
  git clone https://github.com/MrabdulHannan1/hotel-reactjs.git
```

Go to the project directory

```bash
  cd hotel-reactjs
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
